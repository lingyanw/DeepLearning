


# Final project for COMP 576: Predicting Complex Cell Behavior Using Deep Predictive Models
## Set up environment :
A docker file and a docker compose file is included in the folder to easily recreate the develop environment needed. To evoke the docker container, it is required to have `nvidia-docker` installed in the workstation.
## Datasets needed:
All datasets needed for reproducing the results are included in the `data` folder.
## Reproduce the results:
Follow the jupyter notebooks in the following sequence to reproduce all the results:
1. GenerateData.ipynb
2. Training.ipynb
3. Testing.ipynb
